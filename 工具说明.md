# 拼豆图纸工具 — 工具说明

## 一、这个工具是啥？

这是一个 **拼豆（Perler Beads）图纸生成网站**，主要做两件事：

1. **消耗多余豆子**（首页入口）：你选好手头已有的豆子颜色，系统用这些颜色生成拼豆图纸或把上传的图片转成只用这些颜色的图纸，方便你消化余豆、减少浪费。
2. **图片转拼豆图**（原项目能力）：上传任意图片，按拼豆色板像素化、可调参数，导出带网格和色号统计的图纸及采购清单。

整体目标：**把图片变成「按格、按色」的拼豆底稿，并尽量贴合你已有的豆子颜色。**

---

## 二、可以实现什么？

- **用「你选的豆子颜色」出图**：图纸里的颜色只来自你勾选的那套色，方便按现有库存拼。
- **AI 生成图案**：选色后一键生成多套卡通风图纸（需配置 API），或不上传图片也能出图。
- **上传图片还原为拼豆图**：上传一张图，用当前选择的颜色尽量还原原图，得到可拼的格子图。
- **导出可用的拼豆图纸**：带网格、坐标、色号、各色颗数统计，并可导出 CSV 等，便于照着拼和采购。

## 三、具体功能列表

### 1. 消耗多余豆子（主流程，入口：/consume-beads）

| 功能 | 说明 |
|------|------|
| **选择豆子颜色** | 从色板中勾选你已有的颜色；支持多种色号系统（如 MARD 等）；可全选 / 清空。 |
| **颜色预设** | 把当前选择保存为预设（如「我的余色 1」），下次一键应用；应用后可再勾选/取消部分颜色并「更新」该预设；支持删除预设。 |
| **AI 生成图纸（3 套）** | 根据当前选色，从 7 类主题（动物类、人物类、游戏/像素类、食物类、植物类、日常物品类、场景/风景类）中随机抽 3 个，调用图像生成 API 出图，再像素化为拼豆图。需在环境中配置 API（如 Google Gemini）。 |
| **上传图片转图纸** | 上传一张图，用当前选择的颜色把图「还原」成拼豆图（尽量贴近原图色彩）；横轴默认 50 格；背景自动留空，只保留主体连通区域。 |
| **图纸展示** | 每套图纸以一张带网格的拼豆图展示，并显示尺寸、总豆数、使用颜色数、各色统计。 |
| **下载图纸** | 使用与原项目一致的「下载图纸设置」弹窗：可勾选网格线、网格间隔、坐标数字、格内色号、色号统计等，导出 PNG；可选同时导出 CSV 源数据。 |

### 2. 图片与图纸处理逻辑

| 功能 | 说明 |
|------|------|
| **横轴 50 格** | 生成/上传转图纸时，默认横轴 50 格，纵轴按比例，保证格子规整。 |
| **背景不填色** | 自动识别与四边连通的背景并标为「不填色」，图纸上这些格为空。 |
| **边缘密封** | 只保留与主体连通的一块颜色，去掉零散孤立格，方便实际拼豆时豆子连在一起。 |
| **调色板自动加黑白** | 除你选的颜色外，自动加入黑色和白色，用于描边或高光等。 |
| **上传图尽量贴近原图** | 上传图片时用「平均色」模式做像素化，再用当前选色映射，尽量还原原图色彩。 |

### 3. 原项目能力（完整像素化工作流）

从其他入口进入原主页后，仍可使用：

- 上传图片 → 可调粒度（横轴格数）、颜色合并阈值、多种色板。
- 颜色排除与重映射、背景移除、手动着色/修改。
- 悬停查看格子的色号与颜色、导出带 Key 的图纸和颜色统计图、导出采购清单等。

### 4. 其他

- **色号系统**：支持在多种色号体系间切换，便于和实物豆子编号对应。
- **下载样式**：下载的图纸已去掉品牌标题栏、二维码和「图纸来源」等水印，只保留网格、坐标、色号统计和「总计」等必要信息。

## 四、使用流程简述

1. 打开网站，进入「消耗多余豆子」页面（首页会跳转至此）。
2. 选择你已有的豆子颜色（可先应用预设再微调）。
3. 二选一或组合使用：
   - **AI 生成**：点「AI 生成图纸（3 套）」→ 等待生成 → 查看/下载。
   - **上传图片**：点「上传图片」→ 选图 → 系统用当前选色转成拼豆图 → 查看/下载。
4. 在每套图纸处点「下载图纸」，在弹窗里勾选需要的选项后导出 PNG（及可选 CSV）。

---

## 五、技术栈与配置

- **前端**：Next.js（React）、TypeScript、Tailwind CSS。
- **图像处理**：浏览器端 Canvas，像素化、颜色映射、背景与连通区域处理在本地完成。
- **AI 出图**（可选）：需在服务端配置图像生成 API（如 Google Gemini），并可在 `.env.local` 中配置代理等。上传图片转图纸无需 API。
