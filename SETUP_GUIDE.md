# 快速配置指南

## API Key 配置

你的API Key格式正确，可以正常使用。请按以下步骤配置：

### 方法1：使用示例文件（推荐）

1. 将 `.env.local.example` 文件重命名为 `.env.local`
2. 打开 `.env.local` 文件
3. 你的API Key已经填入，直接保存即可

### 方法2：手动创建

1. 在项目根目录创建 `.env.local` 文件
2. 添加以下内容：

```env
OPENAI_API_KEY=your-openai-api-key-here
IMAGE_GENERATION_SERVICE=openai
```

### 重要提示

⚠️ **安全提醒：**
- `.env.local` 文件已经在 `.gitignore` 中，不会被提交到Git
- 不要将API Key分享给他人
- 如果API Key泄露，请立即在OpenAI平台重新生成

### 验证配置

配置完成后：
1. 重启开发服务器（如果正在运行）
2. 访问 `/consume-beads` 页面
3. 选择颜色后点击"AI生成图纸"
4. 如果配置正确，应该能正常生成图像

## 功能说明

### 1. AI生成图纸（需要API Key）
- 每次随机生成3套不同风格的图纸
- 不满意可以点击"重新生成"按钮
- 支持卡通、表情包、漫画等多种风格

### 2. 上传图片转换（无需API Key）
- 上传任意图片
- 自动转换为拼豆图纸
- 颜色自动限制在你选择的颜色范围内
- 完全免费使用

### 3. 颜色限制
- 所有生成的图纸都只使用你选择的颜色
- 确保图纸中的颜色都是你已有的豆子颜色
- 帮助消耗多余的豆子

## 使用流程

1. **选择颜色**：在左侧选择你已有的豆子颜色
2. **生成方式**：
   - **方式1**：点击"AI生成图纸"按钮（需要API Key）
   - **方式2**：点击"上传图片"按钮（无需API Key）
3. **查看结果**：右侧显示生成的图纸预览
4. **重新生成**：如果不满意，点击"重新生成"按钮
5. **下载图纸**：点击每套图纸的"下载图纸"按钮

## 常见问题

**Q: 为什么AI生成需要API Key？**
A: AI图像生成需要调用OpenAI的API服务，需要API Key才能使用。上传图片转换功能完全免费，无需API Key。

**Q: API Key安全吗？**
A: API Key存储在 `.env.local` 文件中，该文件不会被提交到Git仓库。只要不分享这个文件，就是安全的。

**Q: 可以生成多少套图纸？**
A: 每次AI生成会随机生成3套不同风格的图纸。如果不满意，可以点击"重新生成"按钮生成新的3套。

**Q: 上传的图片会限制颜色吗？**
A: 是的，上传的图片会自动转换为拼豆图纸，并且只使用你选择的颜色。系统会自动将图片中的颜色映射到最接近的你选择的颜色。

## 技术支持

如果遇到问题：
1. 检查 `.env.local` 文件是否存在且配置正确
2. 确认开发服务器已重启（配置环境变量后需要重启）
3. 查看浏览器控制台的错误信息
4. 参考 `API_CONFIG.md` 了解详细配置说明
